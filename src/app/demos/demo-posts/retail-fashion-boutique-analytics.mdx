## Power BI Features Showcased

### Basic Features
- **Import mode with Databricks connector** - All business logic handled in SQL views
- **Proper date modeling** - Continuous X-axis on true date (`month_start`)
- **Visual formatting** - Professional titles, units (K), % formats, label precision, grid/row padding
- **Intelligent sorting** - Tables by revenue desc, time series by `month_start`

### Intermediate Features
- **Top N filtering** - Dynamic Top 20 products by revenue
- **Conditional formatting** - Optional data bars on revenue columns
- **100% stacked charts** - Mix/percentage analysis for category performance
- **Visual interactions** - Cross-filtering between different imported tables via relationships

### Advanced Features
- **Report-page tooltips** - Rich, context-aware hover details
- **Drillthrough functionality** - Dedicated Brand page with Back button & "Keep all filters"
- **Decomposition Tree** - AI visual to analyze revenue and returns drivers
- **Many-to-many relationships** - Cross filter direction = Both to unify slicers across Gold views
- **Anomaly detection** - Optional Analytics pane integration on line charts
